# IPTV轮播源

此项目通过其它直播源抓取集成一个轮播直播源。

- 每天自动更新直播，提供在线地址
- 可以直接使用各播放平台自行测试
- 程序自动抓取不能保证都可用，请自行检测

> **声明**：所有播放源均收集于互联网，仅供测试研究学习，不得商用！
